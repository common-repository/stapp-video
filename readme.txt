Wordpress Video Shortcode

== Description ==

 * Responsive optimized display of your videos on all devices
 * Very easy handling, no programming skills required
 * Assign a video file of the appropriate size to each end device type (desktop, tablet or smartphone)
 * Your videos are automatically SEO-friendly & responsive
 * Forum or email support
 * Unlimited lifetime updates
 * Simply insert your videos wherever you want, without being bound by layout guidelines.

Plugin language in German or English. Other languages will follow
